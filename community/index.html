Get Involved

Contribute to Ember.js

The source is hosted at GitHub. Just fork the repository and you're ready to contribute to ember.js.

Bring yourself in! Open a Pull Request!

Be sure to add/extend tests for your bug fix, new feature or enhancement.

Found a bug? Open an Issue!

If you found a bug in ember.js, please let the community know. To file a bug go to the issue tracker and create a new issue. A perfect bug report should contain a clear description of what is not happening and what is your expected behavior. Ideally you provide an test case so its immediately shown what exactly is not working and what is the expected behavior.

Get Help

Ask questions on StackOverflow

StackOverflow is used for QA's. Just tag your question with emberjs or search for questions with that tag and maybe your question has already been asked and answered.

If you have a code specific question be sure to add a JSFiddle which helps to locate the problem you are struggling with. You can use this as a starting point.

IRC Channel

There is an IRC Channel available which can be found at #emberjs.
