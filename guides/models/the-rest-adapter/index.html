The REST Adapter

By default, your store will use DS.RESTAdapter to load and save records. The REST adapter assumes that the URLs and JSON associated with each model are conventional; this means that, if you follow the rules, you will not need to configure the adapter or write any code in order to get started.

URL Conventions

The REST adapter is smart enough to determine the URLs it communicates with based on the name of the model. For example, if you ask for a Post by ID:

1

var post = App.Post.find(1);

The REST adapter will automatically send a GET request to /posts/1.
